Maven plugin for TIBCO BusinessWorks

Enable Java-like lifecycle management in TIBCO BusinessWorks

bw:generate-doc-xml

Full name:

io.github.fastcube.factory.tibco.bw.maven:bw-maven-plugin:1.0.0:generate-doc-xml

Description:

This goal generates an XML documentation file from TIBCO BusinessWorks source files.

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: process-sources.
  • Invokes the execution of the lifecycle phase process-sources prior to executing itself.
  • Invokes the execution of this plugin's goal generate-doc-xml prior to executing itself.
