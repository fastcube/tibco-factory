FastCube TIBCO Factory

A Factory, not a framework.

Introduction

The FC TIBCO Factory is a software factory and must not be thought as a framework.

It will only give tools to develop, build, package, deploy, manage and monitor TIBCO applications and environments, but no actual implementation of functional or technical services (such as ones found in frameworks: logging feature, generic error handling and so on).

However, the FC TIBCO Factory will dramatically improve the development of such a framework!

The FC TIBCO Factory is plugin-based and below is a description of each plugin. Before using a plugin, it is advised to start with FC TIBCO Factory common installation & configuration instructions.


bwMaven

bwMaven is a Maven plugin enabling a Java-like lifecycle in TIBCO BusinessWorks projects. It includes dependency management, EAR & Projlib compilation, advanced configuration management, delivery package generation and much more.

View details »

bw6CodeReview

This Maven plugin will run automatic Code Review on TIBCO BusinessWorks projects. It generates XML/HTML reports of rules passed or failed by the projects and has an open architecture to implement custom rules.

View details »

bwUnit

bwUnit is a Maven plugin enabling a JUnit-like unit tests in TIBCO BusinessWorks projects.

View details »
